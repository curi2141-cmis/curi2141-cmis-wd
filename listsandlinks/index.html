Creating a new webpage

Instructions to making a new webpage in a repository folder

  1. Create a new file in your repository folder by typing 'gedit', space,'filename'.html.
  2. When the file is open, type in the basic code of html.
  3. The text inserted in the title in the headchanges the name of the tab at the top.
  4. Typing <h and a number> would make a tag for writing text. Typing <h1> and text would give you the main title.
  5. After inserting all the text, make sure to save it.
  6. To save to your repository folder, click out of the file and do 'git add' and file name, 'git commit' -m "commit message", and 'git push' to type in your username and password of your github account.
  7. Type sftp 'student username'@cmis-hsteacher
  8. Type 'cd public_html'
  9. Type 'put', space, 'filename' to upload it to the production server
  10. If the file has been uploaded safely, it would appear as '<sftp>'
